# AutoTestCoreHidemium

Auto test core Hidemium - chon file Excel, chay check fingerprint theo profile.

## Yeu cau

- Node.js
- npm

## Cai dat

```bash
npm install
```

## Chay ung dung

```bash
npm start
```

Chay che do dev:

```bash
npm run dev
```

## Cau truc thu muc

```text
src/
  main/       Main process cua Electron
  renderer/   Giao dien va logic renderer
sample/       File mau
```
